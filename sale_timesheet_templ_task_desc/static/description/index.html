Allows create and use template description of task.
This module auto add description for internal of product to the service task description.

Usage

Choose Service in Product Type.


Choose create a task in Product > Invoicing > Service Tracking.


Add Description for Internal in Product > Notes > Description for Internal.


Create Quotations, choose our Product and confirm sale


Open service task.
